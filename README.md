# 💰 Cash Management System

A modern **Cash Management Web Application** built with **Python, Django, HTML, CSS, Bootstrap, and SQLite/MySQL**.  
This project helps users manage income, expenses, transactions, and financial records efficiently through a clean and responsive dashboard.

## 🚀 Features

- 🔐 User Authentication (Login/Register)
- 💵 Income & Expense Tracking
- 📊 Dashboard Overview
- 🧾 Transaction History
- 📂 Category Management
- 📱 Responsive UI Design
- 👤 User Profile Management
- 🛠️ Admin Panel Support

## ⚙️ Installation Guide

### 1️⃣ Clone Repository

```bash
git clone https://github.com/rafi-shoishab/Cash_Management.git
```

### 2️⃣ Go To Project Folder

```bash
cd Cash_Management
```

### 3️⃣ Create Virtual Environment

#### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

#### macOS / Linux

```bash
python3 -m venv venv
source venv/bin/activate
```

### 4️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

### 5️⃣ Run Migrations

```bash
python manage.py migrate
```

### 6️⃣ Start Server

```bash
python manage.py runserver
```

### 7️⃣ Open In Browser

```bash
http://127.0.0.1:8000/
```
